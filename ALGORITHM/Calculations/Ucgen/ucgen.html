Üçgen Olma Kurali

Üçgenlerde bir kenarin uzunluğu, diğer iki kenar uzunluğunun toplamindan küçük ve farkindan büyüktür. Bunu sağlayan üç adet kenar ile üçgen oluşturulabilir. Kenar uzunluklari a,b ve c olan bir üçgenle:
|b - c| < a < b + c
|a - c| < b < a + c
|a - b| < c < a + b
eşitsizlikleri yazilabilir.
